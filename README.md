# [Toriend]()

## 요약
arduino와 jetson nano로 만든 스마트 미러, react, express를 이용한 유,아동 양치질 교육 플랫폼

### 기간
2023.01. ~ 2023.02

### 역할
- express를 이용한 백엔드 제작
- openCV, mediapipe lib을 이용한 실시간 얼굴 좌표 검출과 websocket 기술을 통한 검출 좌표 송수신, javascript를 이용한 이미지 좌표 수정을 이용한 AR 기능 제작

### 기술 스택
Python, openCV, express.js, javascripts, mediapipe, Docker

### 프로젝트 리뷰
- 백엔드 개발을 위한 프레임워크를 고르는 과정에서 각 프레임워크의 특징에 대해 알 수 있었다. Spring Boot, Django, fastAPI, express 중에서, 프로젝트 특성 상 CRUD를 구현하지 않았기 때문에, Spring Boot나 Django를 사용하지 않고 fastAPI나 express를 사용하기로 했고, 새로운 기술을 학습하기 위해 express를 사용하기로 결정하였다.
- 실시간 통신 방법에 대해 학습하던 중, websocket 방식과 server sent event 방식 중에서 커맨드 입력의 가능성이 있어, 양방향 통신이 필요했기 때문에, websocket 방식을 선택하였다. 
- 21년도에 학습했던 openCV를 유용하게 사용할 수 있었다.
- Docker를 이용한 배포를 처음으로 해 보았다.
